# One-to-One Association Between Classes

## What is One-to-One Association Between Classes?

An entity object is associated with only one entity object of another class. This is called a **one-to-one association**.

**OR**

If a class holds only one object of another class, and vice versa, then the entities are said to have a **one-to-one association** with each other.

## How to Model Entity Classes with One-to-One Association into Database Tables?

There are **two ways** to model a one-to-one association:

1. **Direct One-to-One**
2. **One-to-Many as One-to-One**
